---

Strictly follow these Prettier formatting rules for all code generation:
- No semicolons (`semi: false`).
- Use single quotes (`singleQuote: true`).
- No trailing commas anywhere (`trailingComma: "none"`).
- Omit parentheses around single arrow function arguments (`arrowParens: "avoid"`).
- Keep lines within an 80-character maximum width (`printWidth: 80`).
